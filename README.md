# Melisa Tsela — Therapist Website

Professional multilingual website for **Melisa Tsela (Psychologist / Psychotherapist)**, including profile pages, contact form, FAQ, blog, and SEO-ready setup.

## Live Website

👉 https://www.melisatsela.gr/

## What this project includes

- Multilingual website (`el` / `en`) with localized routing
- Home, Bio, Contact, FAQ, Privacy, Blog list, and Blog article pages
- Sanity-powered blog content (headless CMS)
- Responsive UI with smooth animations
- SEO foundations: metadata, sitemap, robots, structured data, canonical/hreflang

## Tech Stack

- **Frontend:** Next.js 16, React 19, TypeScript
- **Styling:** Tailwind CSS 4
- **CMS:** Sanity Studio + Next-Sanity
- **Animation/UI:** Framer Motion, Lottie (`@lottiefiles/dotlottie-react`), Lucide Icons
- **Other:** Portable Text (`@portabletext/react`), Calendly integration
